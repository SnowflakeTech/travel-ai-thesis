# Travel AI Thesis

Hệ thống gợi ý lịch trình du lịch cá nhân hóa sử dụng Agentic AI và RAG.

## 1. Tech Stack

- Backend: FastAPI
- Frontend: Next.js
- UI: Tailwind CSS + shadcn/ui
- Database: PostgreSQL
- Vector Database: Qdrant
- ORM: SQLAlchemy
- Migration: Alembic
- Environment: Windows + VS Code + Anaconda + PowerShell
